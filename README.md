# Vapoursynth Docker Container

## Aim

This is meant to make it easy to run Vapoursynth on cloud GPUs. It will 
include:

- Vapoursynth: NO
- Pytorch: YES
- CUDA: YES
- onnxruntime: NO
- ffmpeg: NO
- Plugins
  - ffms2: NO
  - ESRGAN: NO
  - vs-mlrt (with onnxruntime backend): NO
  - vsrife: NO
- Jupyter: NO
  - Yuuno extension: NO
